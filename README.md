# DB_Medical

## 📌 프로젝트 소개
DB_Medical은 병원 데이터베이스 관리를 목적으로 한  
**의료 정보 관리 콘솔 프로그램**입니다.  
환자, 의사, 진료 정보 등의 데이터를 **CRUD(생성, 조회, 수정, 삭제)** 방식으로 관리할 수 있도록 설계되었습니다.

Visual Studio 환경에서 C/C++ 기반으로 개발되었으며,  
데이터베이스 설계 및 SQL 흐름에 대한 이해를 목적으로 한 프로젝트입니다.

---

## 🛠 사용 기술
- Language: C / C++
- IDE: Visual Studio 2022
- Database 개념 기반 설계
- 파일 기반 SQL 스크립트 관리

---

- `*.txt` 파일:  
  각 테이블별 SQL 쿼리(Insert, Select, Update, Delete) 정의
- `.sln / .vcxproj`:  
  Visual Studio 프로젝트 및 솔루션 파일

---

## ✨ 주요 기능
- 환자 정보 관리 (등록 / 조회 / 수정 / 삭제)
- 의사 정보 관리
- 진료(의료) 정보 관리
- 부서(Department) 관리
- SQL 쿼리 분리 설계를 통한 유지보수 용이성 확보

---

## ▶️ 실행 방법
1. Visual Studio에서 `Db_Medical.sln` 파일 열기
2. 빌드 구성: **Debug / x64 또는 Win32**
3. 실행 (Ctrl + F5)
